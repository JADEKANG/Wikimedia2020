Kind of Media
; Into larger units


  1. 방송 미디어
    2.  : 통신망을 통해 음성, 영상, 문자 등의 정보를 일방적으로 불특정 다수에게 동시에 송신하기 위한 매체
  2. 디지털 미디어
    3.  : 디지털 데이터 형태로 정보를 기록 및 배포하는 매체
  3. 매스 미디어
    4.  : 불특정 다수의 대중에 대한 정보를 전달하는 수단이 되는 라디오, 잡지, 신문, 텔레비전 방송 등의 매체
  4. 멀티 미디어
    5.  : 여러 종류의 정보를 한곳에 모아 복합적으로 처리하는 매체(일반적으로 영상, 음악, 텍스트 등의 조합을 포함)
  5. 소셜 미디어
    6.  : 사회적 상호작용을 통해 퍼지고 전달되는 미디어.(일반적으로 컴퓨터를 매개로 하는 정보 및 의견의 교환을 나타냄)
  6. 1인 미디어
    7.  개인이 다양한 콘텐츠를 직접 생산하고 공유할 수 있는 커뮤니케이션 플랫폼이자, 새로운 형태의 커뮤니케이션 채널

Into Smaller Units
; kind of Mass-Media


  1. 인쇄매체
    2. : 시각적 이미지를 통해 메시지를 전달함 ex. 책, 잡지, 신문
      [특징]
    • 상세한 정보전달 가능
    • 기본적인 정보를 기록, 저장, 전달함
    • 일방향적인 정보 전달
    • 정보 전달의 속도가 느림
  2. 전자 매체
    1. 음성매체
      2. : 청각에 의존하는 전달 매체 ex.라디오, 음반, 녹음기
        [특징]
      • 신속하고 휴대성이 높음.
      • 비교적 저렴한 비용
      • 감성 호소에 용이
      • 정보 전달의 속도가 빠름
      • 시각적 정보 처리 곤란
    2. 영상매체
      3. : 시청각 이미지 전달 매체 ex. TV, 영화, 가상현실(VR,AR)
        [특징]
      • 현장감 있는 정보 제공
      • 오락 기능이 뛰어남
      • 영향력이 가장 높은 매체
      • 정보 전달의 속도가 빠름
    3. 뉴미디어(정보매체)
      4. : 시청각 매체 및 쌍방향 매체 ex. 스마트폰, 컴퓨터, SNS
        특징
      • 쌍방향 매체로서 정보 전달의 상호작용이 가능
      • 기대중도 정보 생산자로 참여 가능
      • 대량의 정보 유통 가능
      • 기존의 미디어 복합적 사용
